Mail Checker

Chrome extension for quick email checking right from the browser toolbar. Dark themed, fast, lightweight.

Download Extension
v2.0 · Chrome · Manifest V3

Quick Access

Check inbox from popup without opening webmail. Cached for instant load.

Dark UI

Clean dark interface. Read/unread indicators, HTML message preview.

Two Providers

Roundcube and FirstMail support out of the box.

Lightweight

No background processes. Under 25KB total. No tracking.

Installation

  1. Download the .zip file above
  2. Extract to any folder
  3. Open chrome://extensions in Chrome
  4. Enable Developer mode (top right)
  5. Click Load unpacked and select the extracted folder
  6. Click the extension icon in toolbar to configure

Supported Providers

Roundcube

Auto-login

Enter email, password and Roundcube URL. Extension handles login automatically.

FirstMail

Session-based

Login on firstmail.ltd with "Memorize session" checked. Extension uses browser cookies.
